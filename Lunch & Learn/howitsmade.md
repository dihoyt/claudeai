## Lessons for Building with AI

### 1. Start with Context
Instead of: "Create a prompt"
Try: "Create a prompt for [specific use case] that will be used to [purpose]"

### 2. Specify Constraints
Mentioning "MS Outlook and OneDrive/SP integration is available" helped the AI:
- Include relevant data sources
- Add integration instructions
- Create realistic examples using that data

### 3. Trust the Process
The user didn't micromanage:
- Didn't specify exact structure
- Didn't provide templates to follow
- Didn't request multiple revisions
- Let AI apply expertise in documentation and prompt engineering

### 4. Iterate When Needed
The conversation shows natural iteration:
- First: General repository structure
- Then: Specific Lunch & Learn content
- Finally: Meta-documentation (this file)
